# Random Song Seed Generator

## What does it do?

It generates a random song seed! If given a groq (NOT GROK!) api key you can 
chat with *llama-3.1-8b-instant* by default or change it to what ever you want
as long as it is in groq.

## Why was it made?

Why not. Just a quick project I made in a few work day evenings. I was bored, ok!

## How to use?

Use as it with python as long as you have requirements. I imagine you have
to install groq and dotenv libraries even if you don't use them as they are
imported regardless. Hence why not use the full features!

### AI features

To use the AI features you need to rename the *.env.template* to *.env* and
change the api key to your own.

for example:
```.env
GROQ_API_KEY=ske_GEI99sghoiusGushe2467IGi
```
THIS IS NOT A REAL KEY BTW!
